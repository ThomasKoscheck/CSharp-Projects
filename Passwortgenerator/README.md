## Passwortgenerator

Kleines Tool um zufällige und sichere Passwörter zu erzeugen. Dabei können Zahlen, Großbuchstaben und Sonderzeichen ein/ausgeschlossen werden.

<br>

| :------------ | :------------------ |
| IDE           | Visual Studio 2017  | 
| Sprache       | C#                  |
| .NET Framework| 4.5.2               | 
| Version       | 1.0.0.0             |
